PCB Languages
=============

Syntax highlighting for various [PCB][1] (Printed Circuit Board) formats.

![Man, don't even ask](preview.png)

Currently included:

* [Gerber Images][2]
* [KiCad PCB Boards][3]
* [KiCad PCB Schematics][3]
* [KiCad PCB Layouts][3]
